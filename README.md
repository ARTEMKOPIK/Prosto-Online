# Просто.Онлайн

Готовый веб‑продукт на React + Vite + Tailwind CSS.

## Быстрый запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Что делать дальше
1. Нажать кнопку **Deploy** (Netlify/Vercel автоматически подхватят настройки).
2. Вставить ключ Groq в появившемся окне **«Настройки»** на сайте.
